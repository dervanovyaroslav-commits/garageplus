# Garage Plus

## Проект сайта для автосервиса Garage Plus

Автосервис специализирующийся на ремонте и обслуживании автомобилей Mercedes-Benz W222.

## Технический стек

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Запуск проекта локально

```sh
# Шаг 1: Установка зависимостей
npm i

# Шаг 2: Запуск сервера разработки
npm run dev
```

## Сборка проекта

```sh
# Создание production-сборки
npm run build

# Предпросмотр production-сборки
npm run preview
```# garageplus
